Latest news. Content only.

Articles lists are sourced from RSS feeds and displayed via Google Material design cards. Each article is then parsed via Readability to display text-only and spread up the reading.

http://adrien-repo.github.io/

External ressources and API usage

Google Feed API is used to get latest articles per source. Google Material Design Lite framework is used to render the website. Readability API is used to parse articles and get text-only content. Icon8 is the source of free flat icons used in the website.
